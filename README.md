# NLP Chatbot using TF-IDF and Logistic Regression

This project implements an NLP chatbot that performs intent classification using TF-IDF vectorization, cosine similarity, and Logistic Regression.

## Features
- Tokenization
- Stopword removal
- Lemmatization
- TF-IDF Vectorization
- Logistic Regression Classification
- Cosine Similarity Prediction
- Model Saving and Loading
- Confusion Matrix Visualization
